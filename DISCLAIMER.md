# 重要声明 / Important Disclaimer

## 项目定位 / Project Positioning

**砸锅-skill** 是一个**职场谈判技巧学习工具**，旨在帮助用户：
- 理解谈判僵局的突破方法
- 学习如何在关键时刻改变游戏规则
- 掌握强硬但理性的谈判技巧

**Pot-Smashing Skill** is a **workplace negotiation skills learning tool** designed to help users:
- Understand methods for breaking negotiation deadlocks
- Learn how to change game rules at critical moments
- Master tough but rational negotiation techniques

---

## ⚠️ 正确使用方式 / Correct Usage

### ✅ 鼓励使用场景 / Encouraged Scenarios

1. **打破僵局** / Breaking Deadlocks
   - 谈判陷入僵局，需要新的突破点
   - Negotiations are deadlocked and need a new breakthrough

2. **保护自己** / Self-Protection
   - 对方得寸进尺，需要设立边界
   - The other party is pushing too far and boundaries need to be set

3. **争取合理权益** / Fighting for Legitimate Rights
   - 在合理范围内争取自己应得的利益
   - Fighting for what you deserve within reasonable bounds

4. **改变不公平规则** / Changing Unfair Rules
   - 现有规则明显不公平，需要重新定义
   - Current rules are clearly unfair and need redefinition

### ❌ 禁止滥用 / Prohibited Misuse

1. **无理取闹** / Unreasonable Troublemaking
   - 没有正当理由，故意破坏谈判
   - Deliberately sabotaging negotiations without valid reasons

2. **威胁恐吓** / Threats and Intimidation
   - 使用非法或不道德手段威胁对方
   - Using illegal or unethical means to threaten others

3. **破坏关系** / Relationship Destruction
   - 为了短期利益破坏长期合作关系
   - Destroying long-term partnerships for short-term gains

4. **情绪失控** / Emotional Outbursts
   - 在情绪激动时使用"砸锅"手段
   - Using "pot-smashing" tactics while emotionally agitated
