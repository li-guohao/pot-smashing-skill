This is an AI Skill focused on workplace negotiation, helping you break deadlocks at critical moments and change the game with hardball tactics.

## 话术模板 / Templates

### 警告型 / Warning
```
"我理解贵方的立场，但我必须提醒，如果继续按这个方向推进，
可能会导致[具体后果]。建议我们重新考虑一下合作的基础。"
```

### 条件型 / Conditional
```
"目前的方案我们无法接受。如果[时间]前不能达成[条件]，
我们将不得不考虑[替代方案]。"
```

### 揭底型 / Cards on Table
```
"既然谈到这个程度，我就直说了。[揭露对方问题]。
我的底线是[明确底线]，如果你们无法接受，那我们就到此为止。"
```

### 掀桌型 / Table Flip
```
"看来我们在根本问题上存在分歧。不如暂停合作，各自重新评估。
如果贵方改变主意，可以联系我。"
```

## 砸锅后重建 / Rebuilding After Smashing

1. **冷却期** / Cooling

## 黄金法则 / Golden Rules

✅ **不打无准备之仗** / Don't fight unprepared  
✅ **一击必中** / One-shot kill  
✅ **留有余地** / Leave room
