# 砸锅 Skill / Pot-Smashing Skill

## 核心原则 / Core Principles

1. **精准打击 / Precision Strike**: 不打无准备之仗
2. **一击必中 / One-Shot Kill**: 出手就要有效果
3. **留有余地 / Leave Room**: 砸锅但不撕破脸
4. **重建秩序 / Rebuild Order**: 破而后立，建立新规则

## 使用场景 / Use Cases

- 谈判陷入僵局时 / When negotiations are deadlocked
- 对方得寸进尺时 / When the other party pushes too far
- 需要重新定规则时 / When rules need to be redefined
- 被欺负需要反击时 / When bullied and need to fight back
- 最后一搏时 / When it's the last resort

## 砸锅等级 / Pot-Smashing Levels

### Level 1 - 警告性砸锅 / Warning Shot
敲山震虎，点到为止

### Level 2 - 局部砸锅 / Partial Smash
针对具体问题的强硬回应

### Level 3 - 全面升级 / Full Escalation
改变谈判格局的重磅出击

### Level 4 - 掀桌重来 / Table Flip
彻底打破现状，重新开始
